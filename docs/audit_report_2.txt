# Mundarija
# 1. Qisqacha mazmuni
## 1.1 Veb-saytga umumiy nuqtai nazar
CyberAI UZ (app.cyberaiuz.workers.dev) - bu xavfsizlik mutaxassislari va tadqiqotchilariga qaratilgan kiberxavfsizlik bo'yicha dastlabki bosqichdagi trening va sun'iy intellekt platformasi. Platforma o'zini sun'iy intellekt yordamida tahdidlarni qidirish, nol bilimga ega bo'lmagan ma'lumotlar va jonli bulutli nishonlar bilan birlashtirilgan elit CTF (Capture The Flag) trening muhiti sifatida ko'rsatadi. U Cloudflare Workers (serversiz chekka infratuzilma) platformasida joylashtirilgan va React/TypeScript/Vite yordamida yaratilgan bitta sahifali ilova (SPA) bo'lib ko'rinadi.
Platforma hozirda bosh sahifada o'zini o'zi e'lon qilganidek, "v2.0 Beta" holatida. Audit tashqi qora quti kuzatuvi orqali amalga oshirildi — hech qanday manba kodi, hisob ma'lumotlari yoki ichki kirish ta'minlanmagan. Qisqacha havola qilingan GitHub ombori 404 (maxfiy yoki o'chirilgan) natijasini berdi, shuning uchun kodni ko'rib chiqish faqat ko'rsatilgan natija va HTTP darajasidagi kuzatuvlar bilan cheklangan.

## 2.2 Navigatsiya arxitekturasi muammosi — muhim UX xatosi

Ommaviy navigatsiya (Bosh sahifa, Konsol, Chat) — 3 ta element, quyidagi joylarda ko'rsatilgan: /, /konsol, /leaderboard, /targets, /threats, /projects, /zkp

## 4.2 Autentifikatsiya va avtorizatsiya
Bu kashf etilgan eng muhim me'moriy muammo:
Barcha "himoyalangan" marshrutlar (/dashboard, /leaderboard, /targets, /console) to'liq mijoz tomonida ko'rsatiladi. Tashqi kuzatuvdan to'g'ridan-to'g'ri /dashboard ga o'tish sahifani kirish oqimiga yo'naltirmasdan yuklaydi.
"Kirish" tugmasi navigatsiyada ko'rinadi, ammo haqiqiy autentifikatsiya oqimi kuzatilmadi. Boshqaruv paneli sahifasida noma'lum vaqt davomida "Yuklanmoqda..." yozuvi ko'rsatiladi, bu autentifikatsiya tekshiruvi navigatsiyadan oldin emas, balki dastlabki renderdan keyin amalga oshirilishini anglatadi.
Agar haqiqiy ma'lumotlar API so'nggi nuqtalari server tomonidan ham himoyalanmagan bo'lsa, bu naqsh (mijoz tomonidagi marshrutni himoya qilish) o'z-o'zidan xavfli hisoblanadi.
Javoblarda Set-Cookie sarlavhalari kuzatilmadi, bu sessiyani boshqarish amalga oshirilmagan bo'lishi yoki butunlay mijoz tomonidagi saqlashga (localStorage/IndexedDB) bog'liq bo'lishi mumkinligini ko'rsatadi.
